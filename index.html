Recommendation system

This was a Spring MVC based web application that crawled over Java wikibooks pages and retrieved the top 10 posts for each user given query. The crawled data was preprocessed by removing stop words, porter stemming the content, indexed using Apache Lucene and ranked using TF-IDF score.

Tech Stack: Java, Jquery, CSS3, Boostrap, Spring MVC, Apache Lucene

Interactive Social Visualization

This was a open social user model based data visualization project that tracked user activities such as scroll, post,comment, upvote, downvote and favorite on StackOverflow.com. It utilized Google Charts for analyzing these actions for multiple users over different time span and provided filters for comparing users and actions. A chrome extension browser was designed for capturing these events.

Tech Stack: Java, Jquery, Javascript, CSS3, Spring MVC, Google Charts

Geospatial Analysis

This project consisted of applying spatial statistics to spatial-temporal big data for identifying statistically significant hotspots pick up locations using Apache Spark and Java. The hotspots were determined using Getis-Ord statistic. The dataset comprised over one billion records representing all Yellow Taxi trips in New York for January 2016

Tech Stack: Java, Apache Spark, Maven, Hadoop
